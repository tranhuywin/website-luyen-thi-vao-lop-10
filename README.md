# "Luyện thi vào lớp 10" Project

## Prerequisite

Make sure your node is version 14.15.0 or above to run this project for security reason.

If not, install [nvm](https://github.com/nvm-sh/nvm) to manage node version.


## Tech Stack

- [React](https://reactjs.org/)
- [Redux](https://redux.js.org/) for State Management
- [Bootstrap](https://getbootstrap.com/) for Built in Style
- [firebase](https://firebase.google.com) for database
- [reactrouter](https://reactrouter.com/web) for router
